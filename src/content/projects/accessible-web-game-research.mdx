This ongoing work focuses on making both web and game experiences more accessible through adaptive interfaces and data-driven accessibility profiling.

## Focus Areas

The research and development spans two complementary tracks: building accessibility-focused applications and studying how interfaces can adapt to individual user needs.

## Adaptive UI

Investigating techniques for interfaces that respond to user preferences and assistive technology usage, reducing barriers for users with diverse abilities.

## Observation-Based Accessibility Profiling

Exploring how observing real interaction patterns can inform accessibility profiles, enabling more personalized and effective accommodations.

## Outcomes

The work contributes practical, accessibility-first patterns for web and game development, grounded in inclusive design principles and WCAG standards.
